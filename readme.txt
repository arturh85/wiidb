== WiiDB ==

Wii Title Database

-- Howto setup development environment --

Step 1: Download and Install Java JDK http://java.sun.com/
        Create the Environment Variable JAVA_HOME pointing to the JDK installation directory
		
Step 2: Download and Install Grails http://www.grails.org/Download
        Create the Environment Variable GRAILS_HOME pointing to the Grails installation directory
		Add $GRAILS_HOME/bin to the PATH
		
-- How to run the test webserver --

Step 1: Go into the project directory
Step 2: Run "grails run-app"
